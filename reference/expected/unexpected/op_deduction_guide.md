```cpp
namespace std {
  template<class E>
  unexpected(E) -> unexpected<E>;
}
```

## 概要
`std::unexpected`クラステンプレートの型推論補助。


## 例
```cpp example
#include <cassert>
#include <concepts>
#include <expected>

int main()
{
  std::unexpected x{1};
  assert(x.error() == 1);
  static_assert(std::same_as<decltype(x), std::unexpected<int>>);
}
```
