```cpp
// expected<cv void, E>部分特殊化
constexpr void emplace() noexcept;
```

## 概要
正常値を保持する。


## 効果
正常値を保持していたら、何もしない。
エラー値を保持していたら、エラー値を破棄して正常値を保持する。


## 例
```cpp example
#include <cassert>
#include <expected>

int main()
{
  std::expected<void, int> x = std::unexpected{42};
  x.emplace();
  assert(x.has_value());
}
```
