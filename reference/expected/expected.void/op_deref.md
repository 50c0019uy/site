```cpp
// expected<cv void, E>部分特殊化
constexpr void operator*() const noexcept;
```

## 概要
正常値(`void`)を取得する。


## 事前条件
[`has_value()`](has_value.md) `== true`


## 戻り値
なし


## 例外

## 例
```cpp example
#include <expected>

int main()
{
  std::expected<void, int> x;
  *x;

  std::expected<void, int> y = std::unexpected{42};
//*y;
  // エラー値を保持する y に対する operator* 呼び出しは未定義動作
}
```
