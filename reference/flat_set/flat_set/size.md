```cpp
size_type size() const noexcept;
```

## 概要
コンテナ内の要素の数を返す。


## 戻り値
`flat_set` クラスが内部で保持している `container_type` を `c` とすると、以下を返す。

```cpp
return c.size();
```
* size()[link /reference/vector/vector/size.md]


## 計算量
定数時間。


## 例
```cpp example
#include <flat_set>
#include <iostream>

int main ()
{
  std::flat_set<int> fs;

  std::cout << fs.size() << std::endl;

  fs.insert(1);
  fs.insert(2);
  fs.insert(3);
  fs.insert(1);

  std::cout << fs.size() << std::endl;
}
```
