ヘッダ [`<execution>`](../execution.md) の名前空間 `std::this_thread` では、実行制御ライブラリの一部エンティティを定義する。

```cpp
namespace std::this_thread {
  …
}
```

実行制御ライブラリ全体は名前空間 [`std::execution`](../execution/execution.md) ページを参照のこと。
