## 概要
先頭を後退させた`subrange`を得る。

## 効果

```cpp
auto tmp = *this;
tmp.advance(-n);
return tmp;
```
* advance[link advance.md]

## 例
```cpp example
#include <ranges>
#include <iostream>

template<std::ranges::range R>
void print(const R& r) {
  for (int x : r) {
    std::cout << x << ',';
  }
  std::cout << '\n';
}

int main()
{
  constexpr int a[] = {1, 2, 3, 4, 5};
  std::ranges::subrange sub(a + 3, a + 4);

  print(sub);
  print(sub.prev());
  print(sub.prev().prev());
}
```

### 出力
```
4,
3,4,
2,3,4,
```
