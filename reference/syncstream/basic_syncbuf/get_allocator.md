# get_allocator
* syncstream[meta header]

```cpp
allocator_type get_allocator() const noexcept;
```

## 概要
アロケータのコピーを取得する。


## 戻り値
コンストラクタまたは代入演算子によって設定されたアロケータのコピーを返す。


## 例外
投げない。


## 例
```cpp example
#include <syncstream>
#include <iostream>

int main()
{
  std::osyncstream bout{std::cout};
  bout << "Hello, World!";
  auto alloc = bout.rdbuf()->get_allocator();
}
```
